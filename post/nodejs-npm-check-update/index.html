检查项目可升级的包

  • Current 是当前安装的版本。
  • Wanted 是根据semver 范围内的软件包的最大版本。
  • Latest 是在 npm registry 中被标记为最新的软件包版本。

npm outdated

安装检测插件
npm install -g npm-check-updates

  • Red (显示红色) = major (主版本，或者说是大版本)
  • Cyan (显示青色) = minor(次要版本)
  • Green(显示绿色) = patch (补丁版本)

检测版本并更新package.json
npm-check-updates -u

删除node_modules目录 重新install依赖包
